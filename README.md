colorado-river
==============

[explanatory paper](https://docs.google.com/document/d/13xb4gsoNGpqxpm49TieC3vqXYqWyMIpB81_3muw7jFc/edit)

To install:

1. npm install -g typescript
2. npm install -g http-server
3. brew install gdal

To run:

1. $ make
2. $ http-server build
3. visit URL http-server is listening on
